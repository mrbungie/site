La **Regresión Lineal** busca encontrar la "mejor línea" que se ajuste a un conjunto de puntos. Es la base de gran parte del Machine Learning moderno y la herramienta principal para la predicción de valores numéricos.

## La ecuación fundamental

La regresión lineal busca modelar la relación entre una variable dependiente <Math formula={"y"} /> y una variable independiente <Math formula={"x"} /> mediante la siguiente forma:

<Math formula={"y = \\beta_0 + \\beta_1x + \\epsilon"} block />

Donde:
- <Math formula={"\\beta_0"} />: Intercepto (valor de $y$ cuando $x=0$).
- <Math formula={"\\beta_1"} />: Pendiente.
- <Math formula={"\\epsilon"} />: Término de error aleatorio.

El algoritmo OLS minimiza el **Error Residual (SSE)**:

<Math formula={"SSE = \\sum_{i=1}^{n} (y_{i} - \\hat{y}_{i})^2"} block />

Y el coeficiente de determinación <Math formula={"R^2"} /> se define como:

<Math formula={"R^2 = 1 - \\frac{SSE}{SSTotal}"} block />
